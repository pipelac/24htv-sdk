# 24TV Control Panel

Центр управления API 24часаТВ для технической поддержки оператора связи.

## Быстрый старт

### Вариант 1: PHP Dev Server

```bash
cd panel
php -S localhost:8080
```

Откройте http://localhost:8080 в браузере.

### Вариант 2: На вашем веб-сервере

Скопируйте содержимое папки `panel/` на ваш веб-сервер с поддержкой PHP. Прокси (`proxy.php`) необходим для обхода CORS-ограничений.

## Использование

1. Откройте `index.html` в браузере
2. Введите **Base URL** вашего API (по умолчанию: `https://provapi.24h.tv/v2`)
3. Введите **API Token** вашего провайдера
4. Используйте sidebar для навигации по сервисам

## Файлы

| Файл | Описание |
|------|----------|
| `index.html` | HTML-оболочка с подключением шрифтов |
| `style.css` | CSS дизайн-система (shadcn-inspired, dark/light) |
| `app.js` | Лочгика приложения, реестр всех методов API |
| `proxy.php` | PHP-прокси для запросов к API (обход CORS) |

## Безопасность

- API-токен хранится **только** в `localStorage` вашего браузера
- Токен передаётся **только** через `proxy.php` на ваш API-сервер
- Прокси не логирует и не сохраняет токены

## Требования

- PHP >= 5.6 (для `proxy.php`)
- Современный браузер (Chrome, Firefox, Edge, Safari)
- Расширение cURL для PHP
